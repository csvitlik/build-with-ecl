# Abstract

Helper script to build standalone executable programs with
[Embeddable Common-Lisp](https://common-lisp.net/project/ecl/).

# Use

    $ cd ~/common-lisp
    $ git clone https://github.com/csvitlik/build-with-ecl
    $ cd build-with-ecl
    $ ecl -load build-with-ecl.lisp
    $ ./build-with-ecl test
    $ ./test
